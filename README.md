# ARchemy: recolecta setas y crea pociones desde donde quieras

**ARchemy** es un prototipo de Realidad Aumentada desarrollada con Unity y AR Foundation. Con él, podras transformar cualquier espacio en un bosque, y recoletar setas y crear pociones con ellas-
---

## Funciones

###  1. ¡Busca setas! 🍄
The app scans your physical room (floors, tables, shelves) to "grow" different species of magical mushrooms and plants. Using **AR Plane Detection**, every session is unique to your environment.

###  2. Recolecta las setas 🖐️
Simplemente con un click, podras guardar la seta que elijas en tu inventorio. Podras acceder siempre que quieras al inventorio para aprender más de ellas y para usarlas posteriormente en la creación de pociones.

###  3. Los elementos no se moverán - Persistencia Espacial 📍
¡No te preocupes de perder objetos si te mueves! Aún si te mueves en el espacio, las setas no van a desaparecer. Gracias a **AR Achors** la aplicación recordará donde está cada objetos, así que no te preocupes por ellos.

###  4. Crea pociones *(no disponible)* ⚗️
Con las setas recolectadas, y guardadas en el inventorio, podrás crear pociones siguiendo las recetas del recetario.

---

## Datos técnicos

* **Engine:** Unity 6000.0.50f1
* **AR Framework:** AR Foundation (ARCore para Android)
* **Sistema de Input:** Touch Screen, tocar la pantalla.
* **Rendering:** Universal Render Pipeline (URP) para la optimizavión en moviles.
* **Persitencia:** AR Anchor Manager y Plane Subsystems.

---

## Antes de empezar a jugar:

### Prerequisitos
* Dispositivo Android con soporte **ARCore**.
* Android 11.0 (API level 30) o superior.
* Tener Unity Hub instalado.

### Installation
1.  Clona el repositorio:
    ```bash
    git clone https://github.com/AndreaCVD/ARchemy
    ```
2.  Abre el proyecto en **Unity**.
3.  Asegurate que los **Build Settings** esten para Android.
4.  Haz Build and Run en tu dispositivo y diviertete.

---

**--> Como jugar**
1.  **Explora:** muevete por el espacio para encontrar diferentes setas.
2.  **Interact:** recoje las setas que encuentres, y aprende de ellas gracias a la información de tu inventario.
3.  **Crea pociones:** *(no disponible)* con los ingredientes recojidos, crea pociones siguiendo tu recetario.
